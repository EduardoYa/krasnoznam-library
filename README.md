# Сайт библиотеки с. Краснознаменное

## Запуск локально

```bash
pip install flask
python app.py
```

Открой: http://localhost:5001

## Страницы

| URL | Описание |
|-----|----------|
| `/` | Главная с последними новостями |
| `/news` | Все новости |
| `/news/<id>` | Полная новость |
| `/schedule` | График работы |
| `/about` | О библиотеке |
| `/login` | Вход для библиотекаря |
| `/admin` | Панель управления |
| `/admin/add` | Добавить новость |
| `/admin/edit/<id>` | Редактировать |

## Деплой на Railway

1. Загрузи папку в GitHub репозиторий
2. Подключи в Railway → New Project → GitHub
3. Добавь Variables:
   - `ADMIN_PASSWORD` = придумай пароль
   - `SECRET_KEY` = любая случайная строка
4. Добавь Volume → Mount Path: `/data`
5. Добавь Variable: `DB_PATH=/data/library.db`

## Для библиотекаря

Пароль входа задаётся в переменной `ADMIN_PASSWORD`.
По умолчанию: `library2026`

Как добавить новость:
1. Зайти на /login
2. Ввести пароль
3. Нажать «Добавить новость»
4. Заполнить заголовок и текст (русский обязательно, казахский по желанию)
5. Загрузить фото если есть
6. Нажать «Опубликовать»
